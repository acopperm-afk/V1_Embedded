# V1_Embedded
This is demonstrating Root Access and its ability to take stm32 schematics 

# Servo
This is a Micro Servo 9g 
It has three cables:
1. Brown goes to ground (the one above Vn) on CN6
2. Red goes to 5v since it needs a 5v power supply on CN6
3. Yellow (which is secretely orange) goes to PWN/D9 on CN9
# Bad microphone 
It has four cables:
1. D0 goes to D2 which is on CN9 
2. Voltage goes to 3V3 on CN6
3. Ground CN6 but the 3rd pin from the bottom CN6
4. A0 goes to D2 (3rd up from bottom) on CN9
